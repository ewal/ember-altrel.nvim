# ember-altrel.nvim
Neovim plugin for opening related files. Supports project mix of typescript and javascript.

## Installation

```lua
-- lazy.nvim
{
  'ewal/ember-altrel.nvim',
  opts = { },
  -- lazy = false, -- make command(s) available at startup
}
```

## Commands
Rotate between related files
```
:EmberRelNext
```
